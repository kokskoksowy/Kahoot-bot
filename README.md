# Kahoot-bot

## Usage linux
```
[ ! -d "Kahoot-bot" ] && git clone https://github.com/kokskoksowy/Kahoot-bot.git
pip install -r Kahoot-bot/requirements.txt
python Kahoot-bot/main.py
```
